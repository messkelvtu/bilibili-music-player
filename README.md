# 🎵 B站音乐播放器

一个现代化的B站音乐下载和播放软件，支持从B站视频提取音频、批量下载合集、自动获取歌词等功能。

## ✨ 功能特点

- 🎥 **B站音频提取** - 从B站视频链接提取高质量音频
- 📚 **批量下载** - 支持合集批量下载
- 🎶 **音乐播放** - 内置现代化音乐播放器
- 📝 **歌词显示** - 自动从多个来源获取歌词
- 🎛️ **播放控制** - 播放/暂停、上一首/下一首、音量控制
- 📁 **音乐管理** - 本地音乐库管理
- 🎨 **现代化UI** - 深色主题，美观易用

## 🚀 快速开始

1. **点击右上角 Fork** 这个仓库到你的账号
2. **进入你的仓库**，点击 "Code" → "Codespaces"
3. **点击 "Create codespace on main"**
4. **等待环境启动**（约1-2分钟）
5. **在终端中运行**：
